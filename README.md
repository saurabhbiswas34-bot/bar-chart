# Progress Bar (React)

Small React project with a minimal custom progress bar component.

## Preview

![Progress Bar UI](./project-preview.svg)

## Features

- Very small code
- Value clamped between `0` and `100`
- Accessible with `role="progressbar"` and ARIA values

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
